# EDA_Nicola_Colusso

Este repositorio contiene el proyecto de Análisis Exploratorio de Datos (EDA) realizado sobre un dataset de la industria cinematográfica. El objetivo principal del proyecto es aprender a realizar un flujo completo de EDA, comprendiendo y describiendo los datos antes de cualquier análisis o modelado más avanzado.

---

## Estructura del repositorio

- **data/**  
  Carpeta que contiene el dataset utilizado para el EDA. No se incluye en el repositorio (`.gitignore`) debido a su tamaño.  

- **notebook/**  
  Contiene el notebook principal con todo el análisis exploratorio, visualizaciones y conclusiones del proyecto.

- **scripts/**  
  Contiene los scripts en Python con funciones útiles para el análisis, organizadas y documentadas:  
  - `01_data_cleaning.py`  
  - `02_visualizations.py`  


---

## Objetivo del proyecto

- Realizar un flujo completo y ordenado de EDA sobre un dataset real de cine.  
- Limpiar y transformar los datos para trabajar con un conjunto fiable y representativo.  
- Analizar la evolución de la producción cinematográfica a lo largo del tiempo, la distribución geográfica de las películas, la eficiencia económica por continente y la recepción por género.  
- Detectar patrones, tendencias y posibles sesgos o limitaciones en el dataset.

---

## Requisitos

- Librerías principales utilizadas:
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `seaborn`
